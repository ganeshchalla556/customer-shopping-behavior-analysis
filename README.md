# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using transactional data (3,900 records, 18 features) to uncover insights into purchasing patterns, customer segmentation, and product performance.

## 🎯 Objective
- Understand customer spending behavior  
- Identify high-value customers  
- Analyze product trends and preferences  
- Support business decision-making  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (PostgreSQL)
- Power BI
- Excel

---

## 📊 Data Processing
- Cleaned missing values (Review Rating imputed using median)
- Standardized column names
- Created new features:
  - Age Group
  - Purchase Frequency
- Removed redundant columns

---

## 🔍 Key Analysis (SQL)

### 1. Revenue by Gender
- Male customers generated higher total revenue than female customers

### 2. High-Spending Discount Users
- Identified customers who used discounts but still spent above average

### 3. Top Rated Products
- Gloves, Sandals, and Boots had the highest ratings

### 4. Shipping Analysis
- Express shipping customers spent slightly more on average

### 5. Customer Segmentation
- Loyal customers: 3116  
- Returning: 701  
- New: 83
